# Congress of Vienna 1816

Workspace for a Victoria 3 mod starting on January 1, 1816 and ending on January 1, 1936.

The `mod/Congress of Vienna 1816` directory is the game-facing mod source. The `planning` directory is for design decisions, source notes, and implementation checklists.

For context handoff or future resumption, start with `planning/99_continuation_roadmap.md`.

Locked decisions:

- Start date: 1816.1.1
- End date: 1936.1.1
- Map geometry: vanilla Victoria 3 state/region/province setup
- Historical changes: ownership, tags, pops, buildings/resources, diplomacy, technology, and setup data only
